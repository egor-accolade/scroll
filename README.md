# Scroll

A React app that detects scroll position and reacts to it.

## Getting Started

### pnpm

```sh
pnpm install
pnpm start
```

### npm

```sh
npm install
npm start
```

## Scripts

| Command | Description |
|---------|-------------|
| `start` | Start the dev server |
| `build` | Build for production |
| `preview` | Preview the production build |
